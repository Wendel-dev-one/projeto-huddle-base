# Huddle Base - Projeto Frontend

Este projeto é uma base para o desafio **Huddle**, focado em praticar habilidades de HTML e CSS, incluindo responsividade e boas práticas de estilização.

## 📋 Sobre o Projeto

O Huddle Base é um projeto desenvolvido para praticar e aprimorar habilidades de estruturação de páginas web utilizando HTML e CSS. O foco está em criar uma página responsiva, seguindo boas práticas de estilização e organização de código. O projeto serve como base para o desafio **Huddle**, permitindo que você personalize e evolua o layout conforme sua criatividade.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica da página.
- **CSS3**: Estilização, reset de estilos e responsividade.
- **Design Responsivo**: Utilização de media queries para adaptação em diferentes dispositivos.

## 🎨 Design

O design do projeto foi inspirado em layouts modernos e minimalistas, priorizando a clareza visual e a experiência do usuário. Os materiais de referência, como wireframes e mockups, estão disponíveis na pasta `src/design/`. As imagens utilizadas estão em `src/images/`. Para garantir consistência visual, siga as especificações do arquivo `style-guide.md`, que define cores, fontes e espaçamentos recomendados.

## 📁 Estrutura de Pastas

```
index.html
README.md
style-guide.md
src/
	css/
		estilos.css
		reset.css
		responsivo.css
	design/
	images/
```

- **index.html**: Página principal do projeto.
- **src/css/**: Arquivos CSS organizados por função:
	- `estilos.css`: Estilos principais.
	- `reset.css`: Reset de estilos para garantir consistência entre navegadores.
	- `responsivo.css`: Estilos para responsividade.
- **src/design/**: Materiais de design (ex: wireframes, mockups).
- **src/images/**: Imagens utilizadas no projeto.
- **style-guide.md**: Guia de estilos com especificações de cores, fontes e espaçamentos.

## ❓Como usar

1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Edite os arquivos CSS conforme necessário para personalizar o layout e estilos.

## 🎯 Objetivo

- Praticar estruturação de páginas HTML.
- Aplicar estilos com CSS, incluindo reset e responsividade.
- Seguir o guia de estilos definido em `style-guide.md`.
